# ims-scheduling
Class exercises on distributed constrain optimization
- Contains an original ADOPT implementation: http://dx.doi.org/10.1016/j.artint.2004.09.003
- Is built over the awesome JADE framework: http://jade.tilab.com
- Provides basic graphic interface for usage
- More information can be found in the presentation included!
